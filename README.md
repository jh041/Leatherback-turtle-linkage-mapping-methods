# Leatherback turtle linkage mapping methods
Here, I go over the methods used in the paper: A Linkage Map for the Leatherback Turtle (Dermochelys coriacea) and new directions for Marine Megafauna Conservation Genomics

In particular, Lep-MAP3 commands and other code used in the Dermochelys coriacea linkage map are available in tutorial format

Special credit goes to coauthor Pasi Rastas, creator of the Lep-MAP3 software suite, for the methodological approaches and the awk scripts used herein.
